Welcome to sixth form. We return to the test harness with this form. Instead of writing tests around code which is data-in/data-out, aka synchronous code, we are testing code which passes through asynchronous behaviors. The primary focus is around testing code which uses promises. Much like the code introduced in 5th form, some of the code in the Async Test Dummy form uses callbacks, which means some tests will need more reflection and care as you write them.
